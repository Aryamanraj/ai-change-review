# Contributing

Thanks for improving AI Change Review.

## Development

```bash
npm ci
npm run compile
npm run lint
npm test
npm run package
```

Press `F5` in VS Code to launch an Extension Development Host. Use a throwaway workspace when testing restores, creates, and deletions.

## Pull requests

- Keep each change focused.
- Do not add telemetry, source uploads, model calls, or destructive Git operations.
- Add or update tests for behavioral changes.
- Run the checks above before opening a pull request.
